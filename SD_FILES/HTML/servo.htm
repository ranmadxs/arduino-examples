YaiOS Servo API

Servo Direction Enumeration

Name Value Description
SERVO_DIRECTION_VERTICAL 20001 Mover Servo Vertical
SERVO_DIRECTION_HORIZONTAL 20002 Mover Servo Horizontal
SERVO_DIRECTION_ALL 20003 Mover todos los servos en todas las direcciones (Horizontal y Vertical)

Servo Move Enumeration

Name Value Description
SERVO_CLOCKWISE 21002 Mover Servo en sentido Horario
SERVO_COUNTER_CLOCKWISE 21001 Mover Servo en sentido antihorario

SERVO_ACTION_CONTINUOUS (ID: 200001)

GET / POST

SERIAL

Comando que nos entrega información del dispositivo de red.

Parameter Value Description
P1 Component Type Enumeration Indica el tipo de servo
P2 int Tiempo en [ms] en ejecutar la acción del servo
P3 Servo Direction Enumeration Indica el servo que ejecuta la acción;
P4 Servo Move Enumeration Indica el sentido del movimiento del servo
P5 int Grados que recorre el servo cada 15 [ms]
P6 None
P7 None

Try it out!

curl -X GET --header "Accept: application/json;charset=UTF-8" "http://${clientIp}/cmd?COMMAND=200001&P1=2001&P2=0&P3=20001&P4=21001&P5=2&P6=None&P7=None"

Http Request

http://${clientIp}/cmd?COMMAND=200001&P1=2001&P2=0&P3=20001&P4=21001&P5=2&P6=None&P7=None
